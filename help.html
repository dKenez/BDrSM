Mission setup

See also the REGBOT wiki page for more details

Mission specification consist of mission lines, each line consist of two (lower case) parts divided by a ':'

drive values : continue conditions (conditions are OR'ed)

e.g.:
vel=-0.2 : dist=1.2, time=6
Drive backwards at 0.2m/s for 1.2 meter or max 6 seconds.

Drive values
VEL is velocity in m/s - positive is forward.
ACC is acceleration limit in m/s2.
TOPOS is target position for this missionline + is forward (vel is max vel).
TR is turnradius in metre - always positive.
EDGER, EDGEL follow Right/Left edge of line at -2..2 (in cm).
WHITE set to 1 if follow-line tape is white, else black.
LOG is log interval in milliseconds.
BAL is balancing, uses last value if omitted.
IRSENSOR is IR-sensor to use for control (1=Wall, 2=Velocity).
IRDist is IR-distance to hold (selsor 1 to wall, sensor 2 to leader).
HEAD sets the reference heading in pose coordinates.
SERVO servo (1..5), PSERVO position (-1000..1000), VSERVO servo speed 0=full,1=slow 300=fast.
LABEL label number that can be used by GOTO.
GOTO is a jump to the label number given. Can be limited by COUNT or any other condition.
EVENT generate an event 0..31, event=0 means stop mission.
CHIRP may be used to make a frequency scan with this amplitude, LOG must be > 3 (samples per cycle).
Continue conditions
DIST driven distance in this mission line - positive meters.
VEL robot velocity (positive is forward).
TURN angle turned (max) - degrees, positive is CCV.
HEAD absolute angle in degrees, can be used with '=' within +/-3 deg, <, > (beware of 180 deg folding).
TIME max time in this mission line - positive seconds
COUNT used with GOTO, GOTO is skipped when count is reached.
XL crossing line, value is 0..20, use '=', '<' or '>'
LV line valid confidence 0..20 (0=no line, increase on detect up to 20), use '=', '<' or '>'
IR1, IR2 distance from side IR sensor.
TILT tilt angle (0 is balance point).
LAST condition that ended last line with a condition.
EVENT test for event.
If no condition, then continues right away.
